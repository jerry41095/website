GraphQL Queries

The following is the only root query available via our endpoint:

package (Package)

Naturally, you query all package information by starting with a package name

Argument Type Description
name String! Name of the package

Package names

Please note that melodyAPI indexes packages by the name that you use in your “import” statements, not the path to the repo. If a repository has no top-level package or it’s named “main”, using its path as a package name will return a not-found error.
